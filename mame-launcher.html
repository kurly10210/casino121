ROM: cmtetris

Combo arcade board — switch between Tetris and Cherry Master slots

Ready to launch MAME emulator

🎮 MAME Controls

Cherry Master (Slot mode):

5 / 6 — Insert coin / Bet

1 / 2 — Start / Spin

Arrow Keys — Hold reels / Navigate

Tetris mode:

← → — Move piece left/right

↓ — Soft drop

Space / Button — Rotate / Hard drop

Esc — Exit game  |  Tab — MAME menu

Use the service/test switch in MAME to toggle between Tetris and slots mode.
